# Installation


## Stable release

To install PEAKsat PYthon Public, run this command in your terminal:

```batch

    $ pip install peakpyp
```

This is the preferred method to install PEAKsat PYthon Public, as it will always
install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this 
[Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) 
can guide you through the process.


## From sources

The sources for PEAKsat PYthon Public can be downloaded from the 
[Github repo](https://github.com/pthemis5/peakpyp).

You can either clone the public repository:

```batch

    $ git clone git://github.com/pthemis5/peakpyp
```

Or download the [tarball](https://github.com/pthemis5/peakpyp/tarball/master):

```batch

    $ curl  -OL https://github.com/pthemis5/peakpyp/tarball/master
```

Once you have a copy of the source, you can install it with:

```batch

    $ python setup.py install
```
